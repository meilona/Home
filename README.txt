This application is to track your friends last location and update your location to your friends.
This application features includes:
1. Home : It's the main menu, displaying maps and your friends last location. If you click the pin location, it will displayed their name and their place. User can check in their location by clicking the maps location manually and post the name of the place. Every 10 minutes, this application automatically update user last location and save the place name as "auto update".
2. Friend List : A menu to search, add, and display your friend list.
3. Profile : A menu to display user profile picture and data. User can change the profile picture too. Below user data, there is a list of user's location history. It can be deleted by long press the feed that you want to delete. User can do logging out too by pressing the log out icon in the upper right.
